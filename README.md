# 🌤️ Weather App

A modern weather web app built using HTML, CSS, and JavaScript.

## 🚀 Features
- 🌍 Search any city
- 💾 Saved locations (localStorage)
- 🎨 Dynamic backgrounds based on weather
- 📱 Fully responsive design
- ⚡ Real-time data using OpenWeatherMap API

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
